library(ggrepel)

ggrepel provides extremely useful tools for plotting text that automatically dodges datapoints.

tribble(
  ~dtl, ~gam, ~gam_lo, ~gam_up, ~label,
     0,    0,       0,       0, "Perfect",
    -1,   -1,      -1,      -1, "Anti-conservative",
    +1,   +1,      +1,      +1, "Conservative",
    +1,   -1,      -1,      -1, "False detection",
    -1,   +1,      +1,      +1, "Missed detection",
  +0.5, +0.5,      -1,      +2, "Inconclusive"
) %>%

  ggplot(aes(dtl, gam)) +
  geom_point() +
  geom_text_repel(aes(label = label))

However, the settings can be rather overwhelming. Key arguments are summarized in the vignette:

Option Default
force 1
direction “both”

arrow NULL

One of the most key arguments is point.padding, which helps prevent text from overlapping the points:

tribble(
  ~dtl, ~gam, ~gam_lo, ~gam_up, ~label,
     0,    0,       0,       0, "Perfect",
    -1,   -1,      -1,      -1, "Anti-conservative",
    +1,   +1,      +1,      +1, "Conservative",
    +1,   -1,      -1,      -1, "False detection",
    -1,   +1,      +1,      +1, "Missed detection",
  +0.5, +0.5,      -1,      +2, "Inconclusive"
) %>%

  ggplot(aes(dtl, gam)) +
  geom_point() +
  geom_text_repel(
    aes(label = label),
    point.padding = 1
  )
