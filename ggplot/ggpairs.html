GGpairs

Zach 2019-12-30

GGpairs helps one create scatterplot matrices, and is provided by the packages GGally.

library(tidyverse)
library(GGally)

Basic usage; plot a few variables.

mpg %>%
  select(year, cyl, drv) %>%
  ggpairs
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.
## `stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

When plotting many variables, the printout can get annoying. Use print() to suppress.

mpg %>%
  select_if(is.numeric) %>%
  ggpairs %>%
  print(progress = F)

ggpairs can take an aes() specification. Make sure you dont select out the column youre trying to visualize!

mpg %>%
  select(displ, year, cyl, drv, cty, hwy, class) %>%
  ggpairs(., mapping = aes(color = class)) %>%
  print(progress = F)
